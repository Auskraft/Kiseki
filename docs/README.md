# Kiseki — документация

Персональная offline-first картотека на Flutter + flutter_bloc. Итерация 1 — домен «медиа» (фильмы/сериалы/дорамы/аниме).

- **[TECH_DESIGN.md](TECH_DESIGN.md)** — технический дизайн: принципы, **канонический реестр имён/enum**, журнал решений (ADR), модель данных (DDL/FTS/миграции), архитектура, спецификация домена «медиа», конвейер картинок, бэкап, сквозные требования. Источник правды.
- **[ROADMAP.md](ROADMAP.md)** — план реализации: MVP-объём, этапы A–G с DoD-чеклистами, вехи M1–M8, будущие домены.

Правило против рассинхрона: имена таблиц/колонок/enum берутся только из [реестра в TECH_DESIGN §2](TECH_DESIGN.md#2-канонический-реестр-идентификаторов). Меняешь имя — меняешь там, потом везде.
